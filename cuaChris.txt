Switch from Ctrl to Alt/AltGr
For text editor map Alt/AltGr as same key

Ctrl-<x> and Ctrl-Shift-<x> are identical in terminals

Shift page up/down doesnt work, captured by xterm

User Super/Windows key for window manager as not recognised by terminal


Text:
Alt-c(backward comp Ctrl-c) = copy
Alt-v(backward comp Ctrl-v) = paste
Alt-x(backward comp Ctrl-x) = cut
Alt-d = delete/yank line
Alt-z(backward comp Ctrl-z) = undo
Alt-Shift-z = redo
Alt-a(backward comp Ctrl-a) = select all
Alt-f = search with regex
Alt-h = replace with regex
Alt-s(backward comp Ctrl-s) = save
Alt-Shift-s = save as


Text Navigation:
up = up
down = down
left = left
right = right
end = end of line
home = start of line
Alt-home = start of file
Alt-end = end of file
Alt-left = start of word
Alt-right = end of word
page down = page down
page up = page up
Ctrl-<above>(backward comp Shift<above>) = turn on selection(Shift page up doesnt work, captured by xterm)
